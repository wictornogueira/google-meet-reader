# Google Meet Reader

### Uma extensão de Google Chrome que realiza a leitura de mensagens do chat.

## Instalação

1. Realize o download da última versão clicando [aqui](https://github.com/wictornogueira/google-meet-reader/releases/latest/download/google-meet-reader.crx).
2. Navegue até a aba `extensões` de seu navegador.
3. Arraste o arquivo `google-meet-reader.crx` até seu navegador.

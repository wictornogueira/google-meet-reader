1. Baixe a última versão do repositório clicando [aqui](https://github.com/wictornogueira/google-meet-reader/archive/master.zip).
2. Descompacte o arquivo `google-meet-reader-master.zip`.
3. Navegue até a aba `extensões` de seu navegador.

5. Clique em `Carregar sem compactação`.
